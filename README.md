Auto Elétrica Dalex ⚡🚛

Site institucional desenvolvido para a Auto Elétrica Dalex, empresa localizada em Rio do Sul/SC, especializada em elétrica automotiva, linha pesada, máquinas agrícolas, tacógrafos e aferições.

📌 Sobre o Projeto

O objetivo deste projeto é modernizar a presença digital da empresa através de um site profissional, responsivo e otimizado para divulgação dos serviços, contato com clientes e fortalecimento da marca na internet.

O site apresenta:

Informações institucionais da empresa
Serviços oferecidos
Contato rápido via WhatsApp
Integração com Instagram e Facebook
Localização via Google Maps
Área para emissão e consulta de certificados
Layout moderno e responsivo

🚀 Tecnologias Utilizadas

React
TypeScript
Vite
TanStack Router
CSS
HTML5
